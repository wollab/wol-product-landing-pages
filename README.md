# Wizards of Learning Product Landing Pages

Static landing pages for Wizards of Learning products.

## Pages

- `index.html` - Wizards of Learning service landing page
- `fish-farmer.html` - Fish Farmer vs Alien Species product page
- `tarot-of-learning.html` - Tarot of Learning product page
- `sustainable-farm.html` - ไร่ยั่งยืน / Sustainable Land product page

## Selling Sheet Standard

Product landing pages in this collection include two document actions:

- **Read Selling Sheet** opens a public, viewer-only Google Doc.
- **Download PDF** uses the Google Docs PDF export URL for printing or forwarding.

Selling Sheets are short value-led product overviews. They are not full rulebooks and should be checked against each product's canonical source material before publication.
